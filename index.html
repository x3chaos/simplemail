SimpleMail Help

SimpleMail v2.x has one catch-all command thus far: /mail

  • /mail help [topic]

  • Access the in-game SimpleMail help

  • /mail send <target> [priority:#] <message>

  • Send a message. Replace # in priority:# with the desired priority.

  • Example: /mail send x3chaos priority:4 Fix the server!

  • /mail get

  • Get your and read all of your unread messages.

Note: items inside <these> are required, whereas items inside [these] are optional.

Changes from v1.0

Total code rewrite

SimpleMail v1.0 literally used only two classes, including the extension of org.bukkit.plugin.java.JavaPlugin. Yeah, it was bad. Not only was it impossible for me to keep up with my own code, it was just set up really stupidly and evolved into a mass of kludges and spaghetti code (see, I had to get that in there somewhere in the new site).

So, I did what any semi-new coder would do: I scrapped the whole damn thing and started over from scratch. It's taken longer and it's still in-dev, but it's ready to be released, thankfully. Expect quiiite a few more updates.
